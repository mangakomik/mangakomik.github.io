18. Apa arti "Ichi Ni San Nya Arigatou" dalam bahasa indonesia


1, 2, 3 Makacih Meong ! ^^

[tex]\:[/tex]

Pembahasan

Angka Dasar

[tex]\:[/tex]

Oke, kali ini kita akan belajar tentang sistem angka dalam Bahasa Jepang. Nah, berikut daftar angka dari 1 - 10 dalam Bahasa Jepang :

Satu = Ichi [ 一 ]Dua = Ni [ 二 ]Tiga = San [ 三 ]Empat = Yon/Shi [ 四 ]Lima = Go [ 五 ]Enam = Roku [ 六 ]Tujuh = Nana/ Shichi [ 七 ]Delapan = Hachi [ 八 ]Sembilan = Kyuu [ 九 ]Sepuluh = Juu [ 十 ]

[tex]\:[/tex]

Penerapannya dalam Anime

1. Kyuubi = Ekor Sembilan

2. Juubi = Ekor sepuluh

3. Ichimaru = Satu kesempurnaan

[tex]\:[/tex]

Kalau untuk angka puluhannya, kita tinggal menambahkan sepuluh (juu) di akhir angka 1 - 9. Misalnya :

• Sebelas = 1 + 10 = Ichi + Juu = Ichi juu

• Dua belas = 2 + 10 = Ni + Juu = Ni Juu

• Tiga belas = 3 + 10 = San + Juu = San Juu

• Empat belas = 4 + 10 = Yon + Juu = Yon Juu

• Lima belas = 5 + 10 = Go + Juu = Go Juu

• dan seterusnya.

[tex]\:[/tex]

Angka Ratusan

Dalam angka ratusan setiap angka dasar kita tambahkan 100 (hyaku), tapi ada juga yang bentuk hyaku dileburkan. berikut daftarnya :

100 = Hyaku 200 = 2 + 100 = Ni + hyaku = Nihyaku300 = 3 + 100 = San + hyaku = sanbyaku400 = 4 + 100 = Yon + hyaku = yonhyaku500 = 5 + 100 = go + hyaku = gohyakudan seterusnya.

[tex]\:[/tex]

Penjumlahan dan Pengurangan

Pokoknya :

[tex]\boxed{\sf{ + \: tasu}}[/tex]

[tex]\boxed{\sf{ - \: hiku }}[/tex]

[tex]\boxed{\sf{ = \: wa }}[/tex]

[tex]\:[/tex]

Contohnya

1 + 2 = 3ichi tasu ni wa san desu400 - 100 = 300yonhyaku hiku hyaku wa sanbyaku desudan seterusnya.

35. Apa bedanya "ni hen bang" sama "ni hen pang"?


Jawaban:

ni hen bang dan ni hen pang memiliki dua arti yang berbeda. arti dari ni hen bang adalah 'kamu sangat hebat'. sedangkan, arti dari ni hen pang adalah 'kamu sangat gendut'.

Penjelasan:

你很棒 （Ni Hen Bang）

- 你 = kamu

- 很 = sangat

- 棒 = hebat

arti： kamu sangat hebat

你很胖 （Ni Hen Pang）

- 你 = kamu

- 很 = sangat

- 胖  = gendut

arti： kamu sangat gendut

Jawaban:

Penjelasan:

secara umum, pola untuk menyusun kalimat dalam bahasa Jepang, sesuai kalimat diatas sebagai berikut:

S + partikel (no) + ushiro + partikel (ni) + objek + partikel (ga) + bentuk masu ( Imasu )

dengan arti perkata:

ari-san : ari : nama orangpartikel no : untuk menunjukkan kepemilikanushiro : dibelakangni : partikel kata keterangan tempatdoni-san - Doni ( objek tunggal )ga : partikel untuk menunjukkan object ( Doni ).bentuk masu ( penegasan kalimat )
