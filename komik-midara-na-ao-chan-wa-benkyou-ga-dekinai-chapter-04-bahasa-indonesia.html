8. Q. Bahasa Jepang dari “Belajar Bahasa Jepang” ...A. Nihongo O Manabu.B. Nihongo O Narau.C. Nihongo O Benkyou.D. Benkyou O Nihongo.​


Bahasa Jepang dari “Belajar Bahasa Jepang" adalah...

A. Nihongo o manabu.

                     

Penjelasan

Pilihan B saya rasa kurang tepat, karena kata 'narau' digunakan jika mempelajari suatu hal dari seseorang. Hal yang dipelajari biasanya juga berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

Contoh kalimat menggunakan 'narau' ↴

   父から空手を習います。

   = Chichi kara karate o naraimasu.

   = Belajar karate dari ayah.

Pilihan C kurang tepat, karena kata 'benkyou' merupakan kata benda. Kalimat tersebut bisa tepat bila penulisannya menjadi 'Nihongo o benkyou suru'. Kata 'suru' digunakan untuk mengubah kata benda 'benkyou' menjadi kata kerja.

Pilihan D jelas salah, karena susunan kalimatnya yang tidak tepat dan kata 'benkyou' bukanlah kata kerja. Harusnya kata 'benkyou' diganti dengan 'benkyou suru', dan pola kalimat yang tepat harusnya "[Objek] o [Predikat]".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Perbedaan 'Benkyou suru', 'Manabu', dan 'Narau' :

・Benkyou suru = "Belajar" yang lebih fokus ke hal-hal yang berhubungan dengan akademik.

・Manabu = "Belajar" yang cakupannya lebih luas dari 'benkyou suru'.

・Narau = “Belajar” yang digunakan ketika jelas ada seseorang yang mengajarkan kita akan sesuatu, dan biasa berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Fungsi partikel を (O) :

Menunjukkan objekMenunjukkan tempat yang dilewatiMenunjukkan lokasi yang menjadi titik tolak dari suatu kegiatanMenunjukkan profesi

12. apakah perbedaan part, season, cour, dan chapter dlm suatu film/komik ?​


Berikut ini penjelasan mengenai perbedaan part, season, cour, serta chapter yang biasa dijumpai pada film, buku atau komik:

Part atau bagian dalam film atau komik merujuk pada subdivisi utama yang membagi cerita menjadi bagian-bagian yang lebih kecil. Contohnya, sebuah film dengan tiga bagian yang berbeda dapat disebut sebagai "trilogi". Bagian-bagian tersebut sering kali memiliki alur cerita dan fokus yang berbeda.Season atau musim dalam konteks televisi atau anime merujuk pada serangkaian episode yang ditayangkan dalam periode waktu tertentu, biasanya dalam per sekian bulan atau bahkan satu tahun. Biasanya satu musim memiliki 12 hingga 24 episode, tergantung pada genre dan durasi episode.Cour atau kouru dalam anime adalah istilah yang digunakan untuk menggambarkan pembagian episode dalam satu musim. Istilah "cour"  awalnya digunakan menggambarkan rentang episode selama satu musim. Setiap kouru biasanya terdiri atas 10 - 14 episodeChapter adalah nama lain 'bab' yang biasa kita jmpai dalam komik atau buku. Chapter merujuk pada bagian-bagian besar cerita yang terpisah satu sama lain. Setiap chapter biasanya memiliki memiliki kisah dan alur cerita tersendiri meskipun tentu masih berhubungan dengan cerita utama.Pembahasan

Secara umum bisa dikatakan bahwa part, season, dan cour mengacu pada pembagian cerita pada media audiovisual. Adapun chapter, lebih cenderung mengacu pada pembagian cerita dalam bentuk media cetak seperti komik dan buku. Meski begitu, beberapa anime juga menggunakan istilah chapter untuk mengacu pada episode dalam serinya.

Diao Chan Dikatakan kecantikan Diao Chan dapat membuat awan-awan menutupi bulan purnama. Maksudnya kecantikan Diao Chan menutupi kecantikan bulan purnama.

Diao Chan adalah pelayan Wang Yun yang telah dianggap seperti anak kandung sendiri. Wang Yun lalu memakai siasat wanita cantik dengan persetujuan Diao Chan sendiri untuk memecah belah Dong Zhuo yang saat itu berkuasa sewenang-wenang dengan Lu Bu, panglima andalan sekaligus anak angkat Dong Zhuo snediri. Secara lengkap, kisah Diao Chan diceritakan dalam San Guo Yan Yi.

Nama Diao Chan sendiri tidak tercatat di dalam sejarah tertulis dan sangat mungkin merupakan cerita rakyat yang dibakukan dalam novel San Guo Yan Yi hasil karya Luo Guanzhong. Dalam sejarah tertulis memang tercatat bahwa Lu Bu memiliki hubungan perselingkuhan dengan pelayan Dong Zhuo, tetapi tidak ada bukti bahwa nama pelayan itu Diao Chan. Bahkan sangat mungkin tidak, karena nama Diao tidak umum dipakai sebagai nama keluarga. Lagipula "Diao Chan" mungkin merujuk ke bulu ekor "Diao" / sable (sejenis musang yang bulunya sering dijadikan perlengkapan pakaian) dan dekorasi giok berbentuk "Chan" / cicada (jengkerik / tenggerek) yang merupakan hiasan topi pejabat era Han.

Jawaban:

Catatan mengenai Diaochan hanya terdapat di dalam novel Kisah Tiga Negara. Tidak adanya ditemukan catatan mengenai Diaochan di dalam buku sejarah resmi, menyebabkan sejarahwan tetap menganggap Diaochan sebagai tokoh fiksi ciptaan Luo Guanzhong, sang penulis novel tersebut.

24. APA bahasa Indonesia Dari bahasa oyasumi nasai minako Chan


おやすみなさい美奈子ちゃん (oyasuminasai Minako-chan). Selamat tidur, Minako.

Pendahuluan:

Dalam materi dasar bahasa Jepang, kita mengenal bab あいさつ (aisatsu) yang artinya adalah sapaan. Contoh sapaan seperti "halo", "selamat pagi/siang/sore/malam", bagaimana kabarmu?", dll. Sapaan dalam bahasa Jepang disesuaikan dengan orang yang diajak bicara. Kalau untuk orang tua, memakai sapaan formal. Sebaliknya, jika seperti teman bisa memakai sapaan informal.

Pembahasan:

おやすみなさい美奈子ちゃん (oyasuminasai Minako-chan). Selamat tidur, Minako.

Terjemahan perkata:

おやすみなさい (oyasuminasai)

おやすみなさい (oyasuminasai) artinya adalah "selamat tidur", atau "selamat beristirahat", yang merupakan salam perpisahan pada waktu malam hari. おやすみなさい (oyasuminasai) dipakai dalam situasi formal, seperti kepada orang yang ditinggikan pangkatnya. Nah, adapun versi pelesetan atau versi singkatnya. 37. quiz!!! Bahasa Jepangnya : "Selamat datang ",apa ya ? Apa fungsi partikel "wa "? onii-chan & Nii-chan tolong di jawab...


1. "Selamat datang" dalam bahasa Jepang adalah ようこそ (youkoso). *lebih rincinya bisa lihat di pembahasan*

2. Fungsi partikel は (wa) diantaranya sebagai penanda subjek maupun juga topik, kemudian dapat berfungsi untuk menyatakan kedua hal yang dianggap berlawanan pada kalimat tersebut.

Pembahasan

Macam-macam frasa "selamat datang" dalam bahasa Jepang

╰➝ Youkoso「ようこそ」: bentuk sapaan formal yang umumnya sapaan ini digunakan ketika berada di suatu perusahaan, tempat-tempat resmi, tempat wisata, ataupun seperti dalam suatu negara/kota/wilayah.

╰➝ Okaerinasai「お帰りなさい」: berarti "selamat datang (kembali)" yang dimana ini merupakan semacam bentuk sambutan dari seseorang ataupun keluarga/kerabat di dalam rumah kepada orang yang telah kembali ke rumah. Frasa ini dapat disingkat menjadi お帰り (okaeri) dengan bentuk versi kasual.

╰➝ Irasshaimase「いらっしゃいませ」: Merupakan jenis sapaan formal, yang biasanya frasa ini umum digunakan ketika berada pada suatu toko yang dimana seorang pegawai/pelayan dalam toko akan menyambut kedatangan pelanggan. Frasa ini juga dapat disingkat menjadi いらっしゃい (irasshai) yang terkesan menjadi sapaan yang kurang formal.

╰➝ Agatte kudasai「上がって下さい」: kalau ini merupakan ungkapan sambutan yang dimana merupakan sebuah sambutan yang berupa sambutan atau mengundang seseorang untuk datang kerumahnya. Termasuk salah satu frasa yang formal.

╰➝ Douzo oagari kudasai「どうぞお上がり下さい」: ini hampir mirip penggunaannya dengan ungkapan 上がって下さい (agatte kudasai) namun yang membedakan adalah bahwa kalimat どうぞお上がり下さい (douzo oagari kudasai) termasuk salah satu bentuk frasa yang terkesan sangat sopan dan memiliki tingkat keformalan yang tinggi.

__________

Partikel wa「は」

Tentunya partikel ini memang sudah tidak terdengar asing dalam beberapa kalimat, partikel itu berupa kata bantu, penulisan semua partikel itu berasal dari huruf hiragana. Namun ada beberapa huruf hiragana ketika dijadikan sebagai bentuk partikel memiliki pelafalan/pengucapan huruf yang berbeda dari pengucapan huruf biasanya.

Seperti salah satunya partikel は (wa) yang dimana cara baca huruf yang sebenarnya adalah ha, namun karena ini dijadikan sebagai bentuk partikel maka pelafalannya diganti menjadi wa.

Kemudian untuk fungsi partikel は (wa) seperti yang sudah saya sebutkan diatas yang diantaranya adalah menjadi bentuk penanda subjek maupun juga topik, kemudian dapat berfungsi untuk menyatakan kedua hal yang dianggap berlawanan pada kalimat tersebut. Namun ada kalanya dalam beberapa konteks dalam kalimat, partikel は (wa) tersebut dapat diartikan "adalah/yaitu."

40. apa arti dari なちゃん身体は平気？na-chan karada wa heiki?​


なちゃん身体は平気？ (Na-chan karada wa heiki?) artinya dalam Bahasa Indonesia adalah "Na tubuhmu ngga apa-apa?".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

KEIYOUSHI「形容詞・けいようし」

Merupakan istilah umum untuk "KATA SIFAT" dalam Bahasa Jepang. Kata sifat dalam Bahasa Jepang dibagi menjadi dua kelompok. Kelompok satu adalah kata sifat-i, yaitu kata sifat yang terdapat tambahan huruf i di akhir kata nya. Dan kelompok dua adalah kata sifat-na, yaitu kata sifat yang tidak terdapat tambahan huruf i di akhir kata nya. Pada kata sifat-na sebenarnya terdapat tambahan huruf Na di akhir kata nya, tetapi ini hanya digunakan saat menerangkan sesuatu saja.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pengertian 平気 (Heiki)

Heiki merupakan sebuah kata sifat-na sekaligus kata benda, yang memiliki arti "Ketenangan", "Ketidakpedulian", "Ketidakacuhan", dan lain-lain. Kata ini juga biasa digunakan sebagai ungkapan "Tak apa-apa", "Tak masalah", dan lainnya yang penggunaannya kurang lebih mirip dengan Daijoubu. Heiki ini lebih biasa digunakan di percakapan kasual.

Contoh penggunaan dalam kalimat :

1. 私は平気です

= Watashi wa heiki desu

= Saya tak apa-apa

2. 平気だよ、心配すんな

= Heiki da yo, shinpai sunna

= Aku gak apa kok, gak usah khawatir

- - - - - - -

平気 (Heiki) x 大丈夫 (Daijoubu)

Dalam kebanyakan situasi, kedua kata ini bisa sama-sama digunakan. Namun terdapat sedikit perbedaan makna pada kedua kata ini. Daijoubu makna nya lebih luas dibanding Heiki. Heiki hanya untuk perasaan saja yang tidak apa-apa, sedangkan Daijoubu bisa untuk apa saja, baik perasaan ataupun keadaan sebenarnya. Heiki digunakan apabila si pembicara ini sebenarnya mendapat masalah, tapi tidak memikirkan atau tidak mempedulikan masalah tersebut. Sedangkan Daijoubu digunakan apabila si pembicara ini memang tidak mendapat masalah.

Kesimpulan :

Kata "Tak apa-apa" pada Heiki mengandung makna bahwa pembicara tidak memikirkan masalahnya walaupun mungkin dia sebenarnya mendapat masalah, sedangkan pada Daijoubu mengandung makna bahwa pembicara memang tidak mendapat masalah.

- - - - - - - - - - - - - -

Penjelasan kosakata lainnya :

なちゃん (Na-chan) = Na

Na di sini merupakan nama orang, sedangkan Chan adalah akhiran panggilan atau gelar kehormatan yang digunakan untuk orang yang sudah akrab (khususnya perempuan).

身体 (Karada) = Tubuh/Badan

Merujuk pada tubuh atau badan secara keseluruhan. Jika ditulis dengan Kanji itu, berarti hanya bisa digunakan untuk tubuh atau badan manusia dan hewan berukuran besar saja.

は (Wa) = Pendanda subjek

Merupakan partikel yang digunakan untuk menandakan subjek atau topik dari sebuah kalimat. Partikel ini ditulis dengan huruf Ha, tapi dibaca Wa.
